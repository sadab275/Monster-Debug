Firstly,The storage capacity of local storage is 5/10 MB. On the other hand, the session storage capacity is 5 MB. Secondly,Local storage can be deleted by JavaScript manually but session storage is session-based and works per window or tab. This means that data is stored only for the duration of a session untill the browser or tab is closed. Moreover, the client can only read local storage. But session storage cannot be read by the client.

In global scope variables declared globally. Outside function global variables can be accesssed from anywhere in JavaScript.A variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop is called block scope.

JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events and executing queued sub-tasks. The event loop is a mechanism that allows JS to perform non-blocking operations. When an asychronus task is started, the event loop will start running. Once the task is completed, the event loop will again check for any other tasks that need to be performed.

There are so many ways to get undefined. Firstly, A declared variable but not yet assigned with a value (uninitialized) is by default undefined. Secondly, When accessing a non-existing object property, JavaScript returns undefined. Thirdly,The function parameters implicitly default to undefined.Fourthly,Implicitly, without return statement, a JavaScript function returns undefined.
